ELIXIR_ITA course application process:

You will need to send us 3 documents in MS Word or PDF formats.
  1. Fill an Application form using the template available in MS Word or pdf formats, and save it using your name as part of the file name: Surname_Name_AF_{{ course.code }}.pdf (PDF format preferred)
  2. Prepare a short CV and save it as Surname_Name_CV_{{ course.code }}.pdf (PDF format preferred)
  3. Prepare a short, one paragraph, simple Motivation statement such as: "I, (Name surname) wish to attend the BPBR15 training course because..." and save it as Surname_Name_MS_{{ course.code }}.pdf (PDF format preferred)
  4. Send an e-mail to elixir-ita-training@ibbe.cnr it (subject line: Course on RNA-Seq CINECA Rome) with the 3 documents in attachment.

IMPORTANT NOTE: Application e-mails that do not carry the three required documents as attachments will not be considered.
